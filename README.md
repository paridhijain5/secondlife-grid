# SecondLife Grid

SecondLife Grid is an AI-powered circular fashion platform that helps users buy, sell, donate, and upcycle pre-loved clothing while reducing textile waste.

## Features
- AI-powered clothing recommendations
- Sustainable fashion marketplace
- Upcycling and donation support
- Impact tracking dashboard
- Modern responsive UI

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Installation

```bash
npm install
npm run dev
